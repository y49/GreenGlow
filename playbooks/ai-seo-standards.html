AI SEO (AIO/GEO) Playbook

Standards, templates, schemas, and copy-ready phrases for AI-driven and generative search (Google AI Overviews, ChatGPT, Gemini, Perplexity).

1) Standards for AIO/GEO

User Intent

  • Answer top-of-page with concise solutions tailored to queries like “Where to commission contemporary art?”
  • Use long-tail, intent-driven keywords and localized variants.

E-E-A-T

  • Showcase experience (case snapshots), expertise (materials/process), authority (partners), trust (contracts/licensing).
  • Expose Last-Updated and citations to authoritative sources.

Multi-Modal

  • High-quality images with descriptive alt text; short videos/tours for installations.
  • PDF spec sheets for giclée (paper/ink/ppi/archival claims).

Technical

  • Schema.org: WebSite/Organization/LocalBusiness/FAQPage/BreadcrumbList + VisualArtwork/CreativeWork/Event when relevant.
  • Hreflang + x-default; CDN; Core Web Vitals; mobile-first.

5) Alt Text & File Naming

  • Format: year-city-subject-material-purpose.jpg
  • Example: 2025-nyc-lobby-installation-aluminum-archival-mounting.jpg
  • Alt text includes: subject, location, year, material, purpose; avoid “image of”.

6) Hreflang & CDN

  • Hreflang: en (x-default), zh-CN; add en-GB / en-US if region-specific pages exist.
  • CDN: serve images from a global CDN; enable compression and WebP/AVIF where supported.
  • Core Web Vitals: LCP < 2.5s; CLS < 0.1; INP < 200ms.
